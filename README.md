# Habit streak tracker

another terminal based application, used for tracking dalyh habits. 

## What it does

This tool provides a lightweight, text-based menu to log a daily habit and track your overall progress. 
* **Daily Logging:** Records the current date to ensure you only log once per day.
* **Streak Tracking:** Calculates the total number of days you've successfully completed your habit.
* **Local Storage:** Safely reads and writes to a local `data.json` file, with built-in error handling for corrupted data or invalid inputs.
